## TermBlog

一个对搜索引擎极不友好的博客。  
基于express，简洁干净，无数据库。  

### 部署

```bash
git clone https://github.com/youzhiye/TermBlog.git
cd TermBlog
npm install --save 
node index.js or npm start
```  

*注：当前版本为开发体验版本，更多的细节及体验正在优化中。。。*
